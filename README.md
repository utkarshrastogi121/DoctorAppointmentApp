# DoctorAppointmentApp

DoctorAppointmentApp is a modern, fully responsive full-stack web application that streamlines the process of booking and managing doctor appointments. Built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js), the application is tailored for **both patients and doctors**, offering intuitive interfaces and role-based access.

Patients can search and book appointments with doctors based on specialty and availability, while doctors can manage their profiles, availability, and upcoming appointments. The app supports **secure authentication**, **real-time scheduling**, and includes a fully integrated **Stripe payment gateway** for processing consultation fees or booking charges.

Whether it's for clinics, hospitals, or independent healthcare providers, DoctorAppointmentApp offers a lightweight yet powerful platform to handle appointment management with ease and flexibility—**optimized for both desktop and mobile devices**.

## 🚀 Features

### 👤 User Authentication

- Secure login and registration for patients and doctors.
- Role-based access control.

### 🩺 Doctor Management

- Doctors can update their profiles and manage availability.
- Patients can view doctor profiles, including specialization and timings.

### 📅 Appointment Scheduling

- Real-time appointment booking system.
- Doctors and patients can view upcoming and past appointments.

### 💳 Payment Integration

- Stripe is integrated to handle secure and fast payments.
- Patients can make payments while booking appointments.

### 📋 Personalized Dashboards

- Role-based dashboards:
  - **Patients**: View and manage bookings.
  - **Doctors**: Manage schedule and availability.

### 🔐 Security

- Passwords securely hashed using bcrypt.
- JWT-based authentication.
- Protected routes and role-level access control.

### 📱 Responsive Design

- Fully responsive UI optimized for all screen sizes (desktop, tablet, mobile).
- Built with modern UI libraries and mobile-first design principles.
